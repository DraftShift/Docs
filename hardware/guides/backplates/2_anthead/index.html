    Anthead Backplate¶

    DraftShift Design

    Overview¶

    StealthChanger Backplate for the Anthead.

    • Standard flow version is used for the purpose of this guide.
    • The Spacer is not used for the purpose of this guide. Put it aside in a safe place until building the toolhead.

    • Anthead Backplate x1 [SF] [HF] [UHF]

    • Anthead Spacer x1 [Spacer] [Spacer Full]

    • 4mmx12mm rounded pins with M3 internal thread x3 [AliExpress] [Amazon US]

    • M3 Heat Inserts x2

    • M3x6 FHCS screws (Must be ferrous) x2

    • M3x10 SHCS screws x2

    • M3x8 BHCS screws x1

    • 6x3mm tapered magnet with M2x8 FHCS screw (Alternatively 6x3mm magnet to be glued) x1

    • M3x6 BHCS screws x2

    • Optotap Sensor x1 [AliExpress]

    • DraftShift StealthChanger Github
