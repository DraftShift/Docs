# Welcome to StealthChanger
Welcome to the official Documentation for StealthChanger, a Klipper-based tool changing solution for 3d printers.

## Overview
This website will help guide you through the installation, configuration and usage of the StealthChanger system.


## What is StealthChanger
StealthChanger is a Voron Tap style system that can be used on Voron and other front mount printers created by DraftShift Design. StealthChanger enables the tool to be completely removed from the gantry making it proficient in tool changing when combined with the rest of the DraftShift Design ecosystem.
